## Overview

FinConnect needed a modern web platform to power their peer-to-peer lending service. The existing site was slow, hard to maintain, and poorly ranked in search engines.

We built a fully server-rendered Next.js application with incremental static regeneration, ensuring fast page loads and excellent SEO performance. The content is managed through a headless CMS integration with Sanity, giving the team full editorial control.

The platform features a responsive dashboard for both borrowers and lenders, real-time notifications, and JWT-based authentication for secure access.
